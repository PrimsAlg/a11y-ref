Align

While not necessary for accessibility, the align function is one of the most useful formatting tools in PowerPoint. It can be found in two places. On the Home ribbon, Align is a submenu of the Arrange dropdown in the Drawing Section. When an element is selected, it's available in the Arrange section of the appropriate Format contextual tab. If one object is selected, it will align it to the slide based on the option you choose. If multiple objects are selected, it will align them in relation to each other.
